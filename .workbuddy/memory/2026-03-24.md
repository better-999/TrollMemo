## TrollMemo 项目云端编译方案

- 项目：TrollMemo（iOS HUD 网速显示应用），基于 theos 构建，产物为 `.tipa` 文件
- 问题：用户没有 macOS 机器，无法本地编译
- 解决方案：创建了 `.github/workflows/build-manual.yml`，使用 GitHub Actions 免费 macOS runner（macos-14 + Xcode 15.4）手动触发构建
- 构建命令：`FINALPACKAGE=1 make clean package`，支持 default / rootless / roothide 三种 scheme
- 产物下载：构建完成后从 Actions 页面 Artifacts 区域下载 `.tipa`
- 用户还需要：将代码 push 到 GitHub 后，在 Actions 页面手动触发 workflow
